## Lv.1 기초 문법

### 1. 자기소개 출력기

이름, 나이, 사는 도시를 변수에 저장한 뒤 다음 형식으로 출력하라.

```text
안녕하세요. 저는 김사과입니다.
나이는 20살이고, Seoul에 살고 있습니다.
```

조건:
- 변수 3개 이상 사용
- f-string 사용

확인 포인트:
- 문자열과 숫자를 자연스럽게 섞어 출력할 수 있는가

### 2. 입력값 더하기

사용자에게 숫자 2개를 입력받아 합, 차, 곱, 몫을 출력하라.

입력 예:

```text
10
3
```

출력 예:

```text
합: 13
차: 7
곱: 30
몫: 3.3333333333333335
```

조건:
- `input()` 사용
- 입력값은 반드시 숫자형으로 변환

확인 포인트:
- `input()` 결과가 문자열이라는 점을 이해했는가

### 3. 전화번호 출력 포매팅

`010`, `1234`, `5678`을 각각 변수에 저장한 뒤 `010-1234-5678` 형식으로 출력하라.

조건:
- `print()`의 `sep` 옵션을 사용한 풀이 1개
- f-string을 사용한 풀이 1개

### 4. 자료형 판별하기

다음 값들을 변수에 저장하고 각각의 자료형을 출력하라.

```python
10
3.14
"Python"
True
```

조건:
- `type()` 사용

### 5. 변수 이름 규칙 검사

다음 변수명 중 사용할 수 없는 것을 고르고, 이유를 설명하라.

```text
name
user_age
1score
class
my city
_count
```

확인 포인트:
- 숫자로 시작, 예약어, 공백 규칙을 구분할 수 있는가

조건:
- `split()` 사용
- 딕셔너리와 리스트 활용
- `setdefault()` 사용
- 컴프리헨션 풀이를 최소 1개 포함

### 38. 도서관 대여 시스템

도서와 회원을 관리하는 간단한 시스템을 작성하라.

클래스:
- `Book`
- `Member`
- `Library`

기능:
- 책 추가
- 회원 추가
- 책 대여
- 책 반납
- 대여 가능한 책 목록 출력
- 회원별 대여 목록 출력

조건:
- `Book`에는 제목, 저자, 대여 여부 저장
- `Member`에는 이름과 대여 목록 저장
- `Library`가 전체 흐름 관리
- 없는 책을 대여하려 하면 예외 처리
- 이미 대여 중인 책은 다시 대여 불가
- 객체 출력용 `__str__()` 구현

추가 도전:
- 책 제목으로 검색할 때 대소문자 무시
- 대여 목록을 순회할 수 있도록 `__iter__()` 구현
