# KDT

## 랭체인을 활용한 AI 영상객체탐지분석 플랫폼 구축과정

### 커리큘럼 6단계 로드맵 (6 Month)
1. 파이썬 기초, 데이터베이스
- 파이썬을 선택한 이유는 AI를 다루기 쉽기 때문
2. 웹서버 프로그래밍 데이터 수집
- AI의 학습을 위해서 데이터가 많이 필요하기 때문에 데이터 수집은 중요
3. 데이터 분석, 머신러닝 / 딥러닝 기초
- 머신러닝 / 딥러닝 기초와 기계를 도메인 지식에 기반해서 효율적으로 사용하는 방법
4. 자연어 처리, 컴퓨터 비전
- NLP는 사람이 하는 언어를 읽고, 이해하며 읽을 수 있게 하는 것
- 컴퓨터 비전은 사람이 본것을 보고, 이해하고, 표현할 수 있게 하는 것
5. 멀티 모달, 강화학습
- 기계가 스스로 학습을 하고 스스로 진화하고 강화학습을 하게 하는 것을 설계
6. 랭체인 & 랭그래프, Agent
- 랭체인 > 랭그래프, Agent, 토큰을 적게쓰는 방식을 중요시 여김
