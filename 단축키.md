## ipynb에서 자주 사용하는 단축키

Jupyter Notebook이나 VSCode의 `.ipynb` 파일에서는 셀을 빠르게 실행하고 편집하기 위해 단축키를 자주 사용한다.

### 1. 실행 관련 단축키

| 단축키          | 설명                              |
| --------------- | --------------------------------- |
| `Shift + Enter` | 현재 셀 실행 후 아래 셀로 이동    |
| `Ctrl + Enter`  | 현재 셀 실행 후 현재 셀에 머무름  |
| `Alt + Enter`   | 현재 셀 실행 후 아래에 새 셀 추가 |

### 2. 셀 추가 / 삭제

| 단축키  | 설명                      |
| ------- | ------------------------- |
| `A`     | 현재 셀 위에 새 셀 추가   |
| `B`     | 현재 셀 아래에 새 셀 추가 |
| `D` `D` | 현재 셀 삭제              |
| `Z`     | 삭제한 셀 되돌리기        |

### 3. 셀 모드 변경

| 단축키 | 설명               |
| ------ | ------------------ |
| `Y`    | Code 셀로 변경     |
| `M`    | Markdown 셀로 변경 |
| `R`    | Raw 셀로 변경      |

### 4. 편집 모드 / 명령 모드

| 단축키  | 설명                           |
| ------- | ------------------------------ |
| `Enter` | 선택한 셀을 편집 모드로 변경   |
| `Esc`   | 편집 모드에서 명령 모드로 변경 |

### 5. 복사 / 붙여넣기

| 단축키      | 설명               |
| ----------- | ------------------ |
| `C`         | 셀 복사            |
| `X`         | 셀 잘라내기        |
| `V`         | 셀 아래에 붙여넣기 |
| `Shift + V` | 셀 위에 붙여넣기   |

> 참고: `A`, `B`, `M`, `Y`, `D D` 같은 단축키는 셀을 선택한 뒤 `Esc`를 눌러 명령 모드에서 사용한다.
## Markdown에서 자주 사용하는 문법

Markdown은 문서를 간단한 문법으로 꾸밀 수 있는 작성 방식이다.  
제목, 목록, 코드블록, 링크, 이미지 등을 쉽게 표현할 수 있다.

### 1. 제목

````tex
# 제목 1
## 제목 2
### 제목 3
#### 제목 4
2. 굵게 / 기울임
**굵게**
*기울임*
~~취소선~~ 
3. 목록
- 항목 1
- 항목 2
- 항목 3
1. 첫 번째
2. 두 번째
3. 세 번째
4. 코드

한 줄 코드는 백틱 한 개를 사용한다.
`print("Hello")`
여러 줄 코드는 백틱 세 개를 사용한다.

```python
print("Hello, Python")
```
````
5. 인용문
> `>  인용문입니다.`

6. 링크
  `[링크 설명](링크 주소)`

7. 이미지
  `![이미지 설명](이미지 주소)`
