# 좌석 예약 관리 프로그램

다음 좌석 목록으로 좌석 예약 관리 프로그램을 작성하시오.

```python
seats = ["A1", "A2", "A3", "B1", "B2", "B3", "C1", "C2", "C3"]
reserved = []
```

## 요구사항

프로그램이 시작되면 사용자는 메뉴 번호를 입력할 수 있다.

| 메뉴 번호 | 기능 |
| --- | --- |
| `1` | 좌석 예약 |
| `2` | 예약 취소 |
| `3` | 예약 현황 보기 |
| `4` | 프로그램 종료 |

메뉴 선택은 반드시 `match-case` 문을 사용하여 처리한다.

### 1. 좌석 예약

사용자가 `1`을 입력하면 좌석 예약을 처리한다.

- 예약 가능한 좌석 목록을 먼저 출력한다.
- 예약 가능한 좌석은 `seats`에는 있지만 `reserved`에는 없는 좌석이다.
- 예약 가능한 좌석 목록은 `available_seats` 변수에 저장한다.
- `available_seats`는 리스트 컴프리헨션을 사용하여 생성한다.
- 좌석 번호를 입력받는다.
- 좌석 번호는 대소문자를 구분하지 않도록 처리한다.
- 입력한 좌석이 예약 가능한 좌석이면 `reserved`에 추가하고 `"{좌석} 좌석 예약 완료"`를 출력한다.
- 입력한 좌석이 이미 예약된 좌석이면 `"{좌석}는 이미 예약된 좌석입니다."`를 출력한다.
- 입력한 좌석이 좌석 목록에 없으면 `"없는 좌석입니다."`를 출력한다.

### 2. 예약 취소

사용자가 `2`를 입력하면 예약 취소를 처리한다.

- 현재 예약된 좌석 목록을 먼저 출력한다.
- 취소할 좌석 번호를 입력받는다.
- 좌석 번호는 대소문자를 구분하지 않도록 처리한다.
- 입력한 좌석이 예약된 좌석이면 `reserved`에서 제거하고 `"{좌석} 좌석 예약 취소 완료"`를 출력한다.
- 입력한 좌석이 좌석 목록에는 있지만 예약되어 있지 않으면 `"{좌석}는 예약되지 않은 좌석입니다."`를 출력한다.
- 입력한 좌석이 좌석 목록에 없으면 `"없는 좌석입니다."`를 출력한다.

### 3. 예약 현황 보기

사용자가 `3`을 입력하면 예약 현황을 출력한다.

- 현재 예약된 좌석 목록을 출력한다.
- 예약된 좌석 수를 출력한다.
- 남은 좌석 수를 출력한다.
- 예약 가능한 좌석 목록을 출력한다.

### 4. 프로그램 종료

사용자가 `4`를 입력하면 프로그램을 종료한다.

- `"프로그램 종료"`를 출력한다.
- 최종 예약 현황을 출력한다.
- 최종 예약 좌석 수를 출력한다.
- 반복문을 종료한다.

### 잘못된 입력

사용자가 `1`, `2`, `3`, `4` 이외의 값을 입력하면 `"잘못된 메뉴입니다. 다시 입력해주세요."`를 출력한다.

## 출력 형식

메뉴는 다음 형식으로 출력한다.

```text
1. 좌석 예약
2. 예약 취소
3. 예약 현황 보기
4. 종료
```

예약 현황은 다음 형식으로 출력한다.

```text
예약 현황: 예약좌석목록
예약 좌석수: 예약좌석수
남은 좌석수: 남은좌석수
예약 가능한 좌석: 예약가능좌석목록
```

최종 예약 현황은 다음 형식으로 출력한다.

```text
프로그램 종료
최종 예약 현황: 예약좌석목록
최종 예약 좌석수: 예약좌석수
```
