# 카페 주문 시스템

다음 메뉴판을 사용하여 카페 주문 시스템을 작성하시오.

```python
menu = {
    "아메리카노": 3000,
    "카페라떼": 4000,
    "바닐라라떼": 4500,
    "레몬에이드": 4200,
    "치즈케이크": 5500,
    "쿠키": 2000
}

orders = {}
```

## 요구사항

프로그램이 시작되면 사용자는 명령어를 입력할 수 있다.

| 명령어 | 기능 |
| --- | --- |
| `메뉴` | 전체 메뉴판을 출력한다. |
| `주문` | 주문 모드를 시작한다. |
| `주문 완료` | 프로그램을 종료하고 최종 주문 내역을 출력한다. |

`주문`을 입력하면 주문 모드로 들어간다.

주문 모드에서는 `"상품명 수량"` 형식으로 주문을 입력받는다.

- `"종료"`를 입력하면 주문 모드를 끝내고 다시 명령어 입력으로 돌아간다.
- 메뉴판에 없는 상품이면 `"없는 메뉴입니다."`를 출력하고 주문을 계속 받는다.
- 같은 상품을 다시 주문하면 기존 수량에 누적한다.
- 주문 모드를 여러 번 실행해도 이전 주문 내역은 유지된다.
- `주문 완료`를 입력하면 최종 주문 내역과 결제 금액을 출력하고 프로그램을 종료한다.
- 상품 금액이 `30000`원 이상이면 10%를 할인한다.

## 출력 형식

메뉴판은 다음 형식으로 출력한다.

```text
=== 메뉴판 ===
아메리카노: 3000원
카페라떼: 4000원
바닐라라떼: 4500원
레몬에이드: 4200원
치즈케이크: 5500원
쿠키: 2000원
```

최종 주문 내역은 다음 형식으로 출력한다.

```text
=== 최종 주문 내역 ===
상품명: 수량개, 금액원
상품 금액: 총금액원
할인 금액: 할인금액원
결제 금액: 결제금액원
```
